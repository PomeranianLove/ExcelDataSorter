## 概要
Excelの支払・取引データを複数の分類キーで自動仕分けし、
分類ごとのExcelブックを自動生成するVBAツールです。

手動で行っていたフィルタ・コピー・ファイル作成作業を自動化します。

## 特徴
- 2重ループによる種別×支払先コードの総当たり仕分け
- Dictionaryを用いた一意値の高速抽出
- ConfigManagerクラスによる設定の一元管理（遅延・フォルダマッピング等）
- DebugLoggerクラスによるメソッド単位のログ記録
- 保存検証・リトライ処理・失敗サマリーの自動出力

## 備考
本リポジトリはMVP（概念実証）版です。
実運用版はクラス構成・キーロジックを大幅に再設計しています。
固有名詞・業務データは一切含みません。

## 動作環境
- Microsoft Excel（VBA実行可能な環境）
- Windows OS
